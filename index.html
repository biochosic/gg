1. Що таке многокутник? (Основи)

Визначення (ТОЧНО): "Многокутник — це проста замкнена ламана лінія та фігура, обмежена нею".

Елементи: 📍 Вершини, 📏 Сторони, 🌉 Діагоналі

  • 📍 Вершини: Точки, де з'єднуються ланки ламаної.
  • 📏 Сторони: Самі відрізки (ланки ламаної).
  • 🌉 Діагоналі: Відрізки, що з'єднують несусідні вершини.

2. Опуклість (Метод лінійки)

Визначення (ТОЧНО): "Многокутник називається опуклим, якщо він лежить по один бік від будь-якої прямої, що містить його сторону".

4. Властивості площі

  • ✅ Додатність: Площа завжди більше нуля (S > 0).
  • ✅ Рівність: Рівні фігури мають рівні площі.
  • ✅ Аддитивність: Якщо фігуру розбити на частини, загальна площа дорівнює сумі площ цих частин.
  • ✅ Одиниця виміру: Площа квадрата зі стороною 1.

📝 ПРАКТИЧНЕ ТРЕНУВАННЯ

№1: Знайдіть суму кутів опуклого десятикутника.

Формула: 180° * (n - 2)

n = 10; 180° * (10 - 2) = 180° * 8 = 1440°.

№2: Скільки всього діагоналей можна провести в восьмикутнику?

Формула: n * (n - 3) / 2

n = 8; 8 * (8 - 3) / 2 = 8 * 5 / 2 = 20 діагоналей.

№3: Площа квадрата дорівнює 49 см². Знайдіть його сторону.

S = a²; a = √49 = 7 см.

№4: Сума кутів многокутника дорівнює 540°. Скільки у нього сторін?

180 * (n - 2) = 540; n - 2 = 3; n = 5 (п'ятикутник).

№5: Зовнішній кут правильного многокутника 45°. Скільки у нього вершин?

360° / 45° = 8 вершин.

№6: Скільки вершин має многокутник, якщо сума його кутів 900°?

180 * (n - 2) = 900; n - 2 = 5; n = 7.

№7: Яка сторона квадрата, якщо його площа 100 см²?

a = √100 = 10 см.

№8: Чому дорівнює сума зовнішніх кутів опуклого дванадцятикутника?

Сума зовнішніх кутів будь-якого опуклого многокутника завжди 360°.

№9: Скільки діагоналей виходить з однієї вершини шестикутника?

Формула: (n - 3); 6 - 3 = 3 діагоналі.

№10: У многокутника 9 діагоналей. Що це за многокутник?

n(n-3)/2 = 9; n(n-3) = 18; методом підбору n = 6 (шестикутник).
